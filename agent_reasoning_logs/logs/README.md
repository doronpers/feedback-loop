# Agent Reasoning Logs

This folder contains reasoning log entries from coding agents.

## File Naming Convention

`YYYY-MM-DD_<repo>_<brief-task-description>.md`

## Creating a New Entry

1. Copy the template from `../templates/reasoning_entry.md`
2. Rename with today's date, repository name, and brief task description
3. Fill in all applicable sections
4. Save to this folder
